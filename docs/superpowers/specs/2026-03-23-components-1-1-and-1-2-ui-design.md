# UI 컴포넌트 1-1, 1-2 분리 설계 명세서 (Design Spec)

**작성일:** 2026-03-23  
**주제:** 창업아이템 배경(1-1) 및 목표시장(1-2) 모듈화 컴포넌트 설계

## 1. 개요 (Overview)
기존 단일 구조였던 사업계획서 프론트엔드 UI에서 "1. 문제인식" 파트를 `1-1. 창업아이템 배경 및 필요성` 파트와 `1-2. 창업아이템 목표시장 현황 분석` 파트로 완전하게 독립 분리된 형태의 React 컴포넌트로 분리합니다. 
두 영역 모두 사용자 합의에 따라 텍스트 나열을 지양하고 시각적 전달력이 높은 **하이브리드 V3 모델(반반 그리드 분리, 정량표+데이터 시각화 차트 결합 구조)**을 채택합니다.

## 2. 디자인 및 구조 원칙 (Design Principles)
- **컴포넌트의 물리적 완전 독립**: 폴더 구조를 명확하게 하기 위해 `ProblemBackgroundV1.jsx`(1-1)와 `MarketAnalysisV1.jsx`(1-2)로 개별 분리합니다.
- **100% 한글화 우선 (순수 한국어)**: 영문 비즈니스 용어를 배제하고, 작성되는 UI상의 글자는 물론이며 모든 변수 설명 및 소스 코드의 로직 서술용 **주석까지도 전면 한국어**로 작성합니다.
- **HWP 모노톤 제약 강제 유지**: 기존에 확립한 `index.css` 베이스를 재사용하며 `--hwp-black`, `--hwp-gray` 단색 계열만 사용합니다. 둥근 모서리나 3D 그림자 효과는 원천 배제합니다.

## 3. 컴포넌트별 단위 레이아웃 명세 (Component Specs)

### 3.1 [NEW] ProblemBackgroundV1.jsx (1-1 사업 개요 파트)
**목적:** 연간 버려지는 굴껍데기 패각 폐기물의 사회적 문제(외부적 배경)와 시중 캠핑 착화재의 발암물질 이슈 해결(내부적 솔루션)을 직관적인 좌-우 대립각으로 대비시켜 보여줍니다.
- **구조**: CSS Flex(혹은 Grid) 시스템을 응용한 반반(`hwp-grid-2`) 박스 구조 양분.
- **시각적 요소**: 
  - 좌측 박스: `lucide-react`의 경고류 심볼(선형 아이콘)을 활용해 기존 시장 캠핑 연료가 가진 심각한 리스크 서술.
  - 우측 박스: 왜 해답이 산화칼슘(소성 굴껍데기) 20% 황금 배합인지, 자체 개발 솔루션의 차별성 요약.

### 3.2 [NEW] MarketAnalysisV1.jsx (1-2 시장 데이터 파트)
**목적:** 줄글을 지양하고, 다각화된 데이터 기반 대시보드 화면을 통해 국가지원 심사역 위원들의 신뢰성을 확보합니다.
- **구조**: 최상단에는 HWP 전통 서식에 맞는 한 줄짜리 핵심 비교표(Table)를 얇게 깔고, 하단 넓은 영역에 `recharts` 라이브러리로 구성된 시각화 모듈 배치.
- **데이터 시각화 파트 구성**: 
  - 우상향 보조 차트(`BarChart` 등): 국내 친환경 야외 용품 시장의 성장률과 자본 유입(보조금 및 수익 마진) 극대화 지표 시각화.
  - 특징 비교 인포그래픽: 제품 간의 안전 특성 및 대상 고객(3040 안전 중심 캠핑족) 파이의 압도적 시장성을 도식으로 노출.

## 4. 모듈 연동 계층 전략 (Integration Layer)
### [MODIFY] BusinessPlanV1.jsx (메인 뼈대 변경)
상위 부모(Container) 컴포넌트였던 모놀리식 구조를 해체하고, 오직 새롭게 분기된 두 모듈을 위에서부터 차례로 `import` 하여 깔끔하게 렌더링하도록 인덱트 트리를 재조립합니다.
```jsx
// BusinessPlanV1.jsx (재편성 컨셉)
import ProblemBackgroundV1 from './ProblemBackgroundV1';
import MarketAnalysisV1 from './MarketAnalysisV1';

export default function BusinessPlanV1() {
  return (
    <div className="hwp-document">
      {/* 고정 타이틀 생략 */}
      <ProblemBackgroundV1 />
      <MarketAnalysisV1 />
      {/* 2단, 3단은 추후 필요시 추가 모듈화 */}
    </div>
  );
}
```
