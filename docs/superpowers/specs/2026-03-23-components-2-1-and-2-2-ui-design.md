# UI 컴포넌트 2-1, 2-2 설계 명세서 (Design Spec)

**작성일:** 2026-03-23
**주제:** 창업아이템 실현가능성(2-1 준비현황, 2-2 고도화 방안 및 산출물) 모듈화 설계

## 1. 개요 (Overview)
"2. 실현가능성(Solution)" 파트를 `SolutionPreparationV1.jsx`(2-1)와 `SolutionExecutionV1.jsx`(2-2) 두 개의 독립된 React 컴포넌트로 분리 선언합니다. 사용자가 채택한 **A안(타임라인 진행도 중심 뷰 + HWP 오리지널 산출물 표 100% 원형 복원)**을 바탕으로 정부 가이드라인의 시각적 기대치를 정확히 맞춥니다.

## 2. 규격 및 디자인 원칙 (Design Principles)
- **가이드라인 양식의 절대 엄수**: 이미지로 제공된 정부부처 가이드라인의 파란색 설명 텍스트 요구사항을 완벽히 소화합니다. 특히 `<최종 산출물 작성>` 표는 셀 병합(rowspan) 처리와 헤더 음영을 실제 HWP 뷰어로 보는 착각이 일도록 정밀하게 CSS로 짜맞춥니다.
- **100% 한글 우선 원칙 계승**: 이전 챕터 1의 무채색 단일 톤(Black/Gray 모노톤)과 순수 한국어 변수명/주석 원칙을 유지합니다.

## 3. 컴포넌트 단위 명세 (Component Specs)

### 3.1 [NEW] SolutionPreparationV1.jsx (2-1 파트 담당)
**목적:** "해당 제품 개발/구체화를 위해 본 사업 신청 전까지 기획/추진한 경과"를 쓰라는 지침에 맞춰, 막연한 아이디어가 아닌 '실행력이 준비된 창업자'임을 시각화합니다.
- **구조**: CSS Flex를 활용해 요소들의 지점(1->2->3)을 잇는 가이드라인 횡단형 **가로 진행 타임라인(Timeline Timeline) UI**.
- **세부 내용 (v6 데이터 문맥 반영)**:
  - 1단계: 통영 현지 폐기물 자원화 시설 직접 실사 및 원료 샘플수급망(MOU) 구두 타진.
  - 2단계: 커피박 + 액상 천연 왁스(핫멜트) + 소성 굴패각 가루 이종 폐기물의 결합 R&D 수만 번 진행.
  - 3단계: 탈황 흡착 임계점을 잡고 발열량 저하를 방어하는 '최적의 산화칼슘 20% 황금 배합 비율(마지노선)' 자체 모델 도출 성공.

### 3.2 [NEW] SolutionExecutionV1.jsx (2-2 파트 담당)
**목적:** 20% 배합 기술 등 핵심 성능 요건(문제점 대응)을 기재하고, 가장 중요한 "전체 및 단기 사업단계 최종 산출물" 일정을 못 박습니다.
- **구조**:
  - `상단부`: 제품을 고도화하는 핵심 원리(HOT-MELT 캡슐 효과, 20% 다공성 방어선)을 컴팩트한 리스트박스로 정리.
  - `하단부`: 제공된 정부 HWP 예시 이미지의 넓이와 테두리 두께 속성까지 모방한 **최종 산출물 표** 배치.
- **<최종 산출물 작성(전체 사업단계)> 표 내부 데이터 구성**:
  - `표 구분 1: 단기 (협약 기간 내)`
    - 품목 A: 친환경 캠핑 착화재(다공성 20% 배합 최적화 모델) 시제품 생산 / 1세트 / 완료: ~'26.11.30.
    - 품목 B: KCL(한국건설생활환경시험연구원) 배기가스 등 5대 안전 시험성적서 및 인체 무해 인증 공인 통과 / 1건 / 완료: ~'26.12.31.
  - `표 구분 2: 중·장기 최종 산출물`
    - 품목 C: 대량 양산 설비 라인 안정화 및 프리미엄 향(톱밥 믹스) 하이엔드 제품 베리에이션 출시 / 1세트 / 완료: ~'27.05.31.

## 4. 연동 계층 (Integration Layer)
현재 `BusinessPlanV1.jsx`의 1-1, 1-2 모듈 렌더링 코드 바로 그 아래 공간을 타겟으로 하여, 새롭게 작성될 `SolutionPreparationV1`과 `SolutionExecutionV1`을 Import하여 스택 방식으로 차례로 쌓이도록 병합 연동합니다.
