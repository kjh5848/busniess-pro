# 예비창업패키지 심사 평가 기준

> 사업계획서 작성 전 이 기준을 숙지하여 심사위원이 높이 평가하는 내용을 우선 작성하세요.

## 2025 심사 배점 기준 (100점 만점)

| 평가 항목 | 배점 | 핵심 심사 포인트 |
|---------|------|---------------|
| **문제인식** | 20점 | 시장 실패/기회의 명확성, 데이터 근거, 해결 시급성 |
| **실현가능성** | 30점 | 기술력, 개발 역량, 차별성, 사업비 집행 타당성 |
| **성장전략** | 30점 | BM 타당성, 시장 진입 전략, 확장성, 재무 계획 |
| **팀 구성** | 20점 | 대표자 역량, 팀 완성도, 주요 인력 확보 계획 |

## 섹션별 심사위원 체크리스트

### 1. 문제인식 (20점)

심사위원이 확인하는 것:
- [ ] 시장 규모가 최신 공신력 있는 데이터로 뒷받침되는가?
- [ ] Pain Point가 단순 불편함이 아닌 구조적 문제인가?
- [ ] 타겟 고객이 명확하게 정의되었는가?
- [ ] 기존 솔루션의 한계가 구체적으로 기술되었는가?
- [ ] "왜 지금 이 문제를 해결해야 하는가"가 설득력 있는가?

감점 요인:
- 출처 없는 시장 수치
- 모호한 타겟 ("모든 사람이 대상")
- 문제와 솔루션의 논리적 연결 부재

### 2. 실현가능성 (30점)

심사위원이 확인하는 것:
- [ ] 기술적 실현 가능성이 TRL 등 구체적 지표로 설명되는가?
- [ ] 개발 로드맵이 현실적이고 세분화되었는가?
- [ ] 경쟁사 대비 차별성이 수치로 증명 가능한가?
- [ ] 지식재산권(특허, 저작권) 전략이 있는가?
- [ ] 사업비 집행 계획이 적법하고 세부적인가?

감점 요인:
- "세계 최초/유일" 주장 근거 없음
- 개발기간 과도하게 낙관적
- 사업비 항목이 부적절하거나 불명확

### 3. 성장전략 (30점)

심사위원이 확인하는 것:
- [ ] 수익 모델이 명확하고 지속 가능한가?
- [ ] Unit Economics (고객 획득 비용 vs 생애가치)가 합리적인가?
- [ ] 초기 고객 확보 전략이 구체적인가?
- [ ] TAM→SAM→SOM 순서로 현실적 목표 시장이 설정되었는가?
- [ ] 3~5년 매출 추정이 가정 근거와 함께 제시되는가?

감점 요인:
- "SNS 마케팅으로 입소문" 같은 막연한 전략
- 시장 점유율 비현실적 (Year 1에 20% 점유 등)
- BM 없이 "일단 사용자 모으기"

### 4. 팀 구성 (20점)

심사위원이 확인하는 것:
- [ ] 대표자의 전문성이 아이템과 직접 연관되는가?
- [ ] 핵심 기술 개발 역량을 팀이 보유하고 있는가?
- [ ] 부족한 역량에 대한 채용/협력 계획이 있는가?
- [ ] 팀 시너지(서로 보완하는 역할)가 설명되는가?

감점 요인:
- 기술 창업인데 기술자가 팀에 없음
- 역할 분담 불명확
- 채용 계획이 구체적이지 않음

## 심사위원 페르소나별 강조 포인트

심사위원 유형에 따라 강조점을 달리한다 (`business-plan-evaluator` 스킬 연계):

| 페르소나 | 관심사 | 강조할 내용 |
|---------|-------|-----------|
| 기술 전문가 | 기술 실현가능성 | TRL, 특허, 알고리즘 설명 |
| 투자 심사역 | ROI, 확장성 | Unit Economics, TAM, Exit 전략 |
| 정부 공무원 | 공공성, 규정 준수 | 고용 창출, 사회적 가치, 규정 준수 |
| 창업 멘토 | 실행력, 팀 | 대표자 스토리, 검증 실적 |

---

## 자주 저지르는 실수 TOP 5

1. **출처 없는 시장 수치**: "AI 시장은 1000조원" → "글로벌 AI 시장 $X조 (IDC, 2024)"로 수정
2. **개조식 불사용**: 문단형 서술 → 핵심 포인트 위주 개조식 + 근거
3. **경쟁사 분석 회피**: "직접 경쟁자 없음" 주장 → 간접 경쟁자 포함 정직하게 분석
4. **사업비 추상적 기술**: "개발비 1000만원" → 단가×수량×횟수 명세 형식
5. **팀 역량 자랑 일색**: 학력·경력 나열 → 아이템 성공에 직결되는 역량 스토리 중심
