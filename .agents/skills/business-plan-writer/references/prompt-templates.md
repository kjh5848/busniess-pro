# 섹션별 고품질 프롬프트 템플릿

> 각 섹션 작성 시 아래 템플릿을 기반으로 구체적인 내용을 채워 사용하세요.
> `[대괄호]` 안의 내용은 실제 아이템 정보로 교체합니다.

---

## 📌 범용 사업계획서 작성 지시문 (Master Prompt)

```
당신은 한국 예비창업패키지 사업계획서 작성 전문가입니다.

아이템: [아이템명]
아이템 설명: [핵심 기술/서비스 한 줄 설명]
타겟 고객: [1차 고객군]
산업 분야: [바이오/AI/핀테크 등]
현재 개발 단계: [아이디어/PoC/프로토타입/MVP]

다음 규칙을 반드시 따르세요:
1. 모든 수치는 출처를 명시 (예: * 출처: IQVIA(2024))
2. 개조식 문체 사용 (불필요한 영어 단어 자제)
3. 심사위원(기술전문가+투자심사역)이 납득할 근거 포함
4. HWP 스타일 문서 형식으로 작성 (표, 목록 적극 활용)
```

## 1. 문제인식 섹션 프롬프트

### 1-1. 창업아이템의 배경 및 필요성

```
[아이템: {아이템명}]의 배경 및 필요성 섹션을 작성해주세요.

포함 요소:
- 거시적 시장 트렌드 2~3가지 (최신 통계 수치 필수)
- 현재 솔루션의 구조적 문제점 (3가지 이상)
- 문제 해결 시 기대 효과 (정량적 수치 포함)
- "왜 지금 해야 하는가" (시장 타이밍 근거)

형식: 핵심 메시지 박스 → 현황 분석 → 문제점 → 해결 필요성
분량: A4 1.5페이지 분량의 텍스트
```

### 1-2. 목표시장 현황 분석

```
[아이템: {아이템명}]의 목표 시장 현황 분석을 작성해주세요.

포함 요소:
- 국내 시장 규모 (TAM/SAM/SOM, 연도별 성장 추이)
- 글로벌 시장 동향 (CAGR, 주요 플레이어)
- 경쟁사 분석 (3~5개사: 자사 vs 경쟁사 비교표 형식)
- 관련 규제/정책 환경

웹 검색으로 최신 시장 데이터 확보 후 작성
출처 필수: 통계청, 한국바이오협회, IQVIA, McKinsey 등
```

## 2. 실현가능성 섹션 프롬프트

### 2-1. 개발 계획 및 구체화 방안

```
[아이템: {아이템명}]의 실현가능성 섹션을 작성해주세요.

포함 요소:
- 현재 기술 준비도(TRL {현재단계} → 목표 TRL {목표단계})
- 핵심 기술 스택 및 활용 오픈소스 (GitHub 링크 포함 가능)
- 월별 개발 로드맵 (최소 6개월, 마일스톤 포함)
- 기술적 리스크와 대응 방안

형식: 표+텍스트 혼합, 개발 단계는 간트차트 테이블 형식
```

### 2-2. 차별성 및 경쟁력 확보 방안

```
[아이템: {아이템명}]의 차별성 및 경쟁력 섹션을 작성해주세요.

포함 요소:
- 고객 Pain Point ↔ 기술 솔루션 매핑 (테이블 형식, 근거 포함)
- 경쟁 모방 난이도 분석 (특허/데이터/네트워크 효과 기반)
  * 모방 불가 근거: [특허 출원 현황 / 독점 데이터 / 기타]
- 미래 경쟁 우위 강화 전략 (Moat 심화)

심사위원이 "왜 이 팀만 할 수 있는가"에 답하는 내용 포함
```

### 2-3. 사업비 집행 계획

```
[총 지원금: {금액}원]에 대한 사업비 집행 계획표를 작성해주세요.

예산 배분:
- 재료비: [금액]원 (단가×수량 명세)
- 외주용역비: [금액]원 (세부 항목별)
- 기타운영비: [금액]원

fund-execution-guidelines 스킬 기준 준수
각 항목: 용도, 단가, 수량, 소계, 집행 시기 포함
금지: 인건비, 개인 사용 물품
```

## 3. 성장전략 섹션 프롬프트

### 3-1. 비즈니스 모델

```
[아이템: {아이템명}]의 비즈니스 모델을 작성해주세요.

포함 요소:
- 주요 수익원 (1차/2차/3차)
- 가격 정책 (경쟁사 대비 포지셔닝)
- 고객 획득 비용(CAC) vs 생애가치(LTV) 추정
- Year 1~3 예상 매출 시나리오 (보수/기준/낙관)

전략 유형: [SaaS/플랫폼/API/라이선스] 중 선택하여 맞춤형 작성
```

### 3-2. 사업화 추진 전략 (시장 진입)

```
[아이템: {아이템명}]의 시장 진입 전략을 작성해주세요.

포함 요소:
- 초기 고객 확보 채널 (구체적인 활동 계획)
  * 1단계: 파일럿 고객 [N]개사 확보 방법
  * 2단계: [채널명] 통해 {기간}내 {목표} 달성
- 파트너십 전략 (협력 기관/기업 리스트)
- 단계별 성장 목표 (분기별 KPI)

마케팅 용어: "GTM" → "시장 진입", "Scale-up" → "성장 단계"로 표기
```

## 4. 팀 구성 섹션 프롬프트

```
[대표자명]의 팀 구성 섹션을 산문체로 작성해주세요.

대표자 정보:
- 학력: [학교/전공]
- 주요 경력: [역할/성과 중심으로 (회사명 최소화)]
- 보유 기술: [관련 기술/역량]
- 아이템과의 연관성: [왜 이 사람이 이 아이템을 해야 하는가]

채용 계획:
- [직군 1]: [채용 시기], [필요 역량]
- [직군 2]: [채용 시기], [필요 역량]

형식: 테이블 대신 유려한 서술형 (심사위원이 팀 스토리를 공감할 수 있게)
분량: A4 1페이지 내외
```

## 5. UI 생성 트리거 프롬프트

텍스트 초안 완성 후 UI 요청 시:

```
위에 작성한 [섹션명] 내용을 HWP 스타일 React 컴포넌트로 만들어주세요.

요구사항:
- hwp-styled-ui 스킬의 Option B (V3 데이터 기반) 적용
- 파일명: [SectionName]V[버전번호].jsx
- Recharts 차트 포함 (흑백 톤 유지)
- 출처 인용 모두 컴포넌트에 포함

먼저 페르소나 랭킹을 제시하고 선택 후 코드 생성
```
