# 프로젝트 스펙 요약 (헤딩 정보)

제공해주신 문서(`사업계획서개요_AI.txt` 등)를 바탕으로, 새 사업계획서 프로젝트 생성에 필요한 7가지 요건을 아래와 같이 추출했습니다.

---

1. **아이템명 (창업 아이템의 공식 명칭)**
   - 슈뢰딩거 FEP+ 대응 오픈소스 기반 고정밀 하이브리드 신약 설계 플랫폼

2. **한줄 소개 (무엇을 하는 서비스인지 1문장)**
   - 초고속 AI 모델(Boltz-2)과 정밀 결합력 예측(OpenFE)을 결합하여 IT 비전문가도 즉각 사용할 수 있는 저비용 고효율 신약 개발 통합 플랫폼

3. **핵심 기술 (주요 기술 스택 또는 방법론)**
   - Boltz-2 (초고속 스크리닝), OpenFE/OpenMM (FEP 정밀 계산), Docker 기반 컨테이너링, GUI 프로토콜 빌더, 소비자용 GPU(RTX 4090) 인프라 최적화

4. **타겟 고객 (1차 고객군)**
   - 고가의 라이선스와 하드웨어 인프라 확보가 어렵고 글로벌 경쟁에서 소외될 위험이 있는 중소 제약사, 초기 바이오 벤처 및 대학/국공립 연구소 연구원

5. **산업 분야 (바이오 / AI / FinTech / SaaS 등)**
   - 바이오 (AI 신약 개발 솔루션 / SaaS & 온프레미스)

6. **개발 단계 (아이디어 / PoC / 프로토타입 / MVP)**
   - 아이디어 및 기획 (2026 하반기 SaaS PoC 베타 출시 목표)

7. **지원사업명**
   - 2025 예비창업패키지 (바이오 분야)
